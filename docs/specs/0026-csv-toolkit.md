# 0026 CSV Toolkit

## 背景

CSV 是开发、测试、运营、产品、数据同学都会频繁接触的通用格式。常见需求包括：

- 粘贴一段 CSV 快速检查列是否对齐
- 在表格视图中确认标题行、空值和异常分隔符
- 把 CSV 转成 JSON 继续给接口、脚本或其他工具使用
- 从 JSON 数组快速导出 CSV 发给业务同学

相比偏开发工具的能力，`CSV Toolkit` 更容易扩大 Magic Box 的用户面，是一个兼顾通用性和 local-first 价值的高频工具。

## 用户故事

作为开发、测试、运营或产品同学，我希望把 CSV 或 JSON 数组粘贴进来后，立刻看到结构化表格预览，并能做基础清洗与格式互转，从而减少临时打开表格软件或写脚本的成本。

## 首版范围

- 输入：
  - 支持粘贴 CSV 文本
  - 支持粘贴 JSON 数组并转成 CSV
  - 支持选择是否有表头
  - 支持设置分隔符：逗号、制表符、分号
- 表格预览：
  - 显示行列数
  - 首版支持基础表格预览
  - 支持空值展示
  - 支持复制单元格或整段结果
- 基础处理：
  - 去除首尾空白
  - 删除空行
  - 统一换行符
  - 按列导出为 JSON 数组
- 格式互转：
  - CSV → JSON
  - JSON 数组 → CSV
  - 支持格式化输出
- 模板：
  - 用户列表
  - 商品列表
  - 埋点报表片段

## 不做什么（首版不含）

- 不做复杂 Excel 公式支持
- 不做 `.xlsx` 文件解析
- 不做海量数据分页和虚拟滚动优化
- 不做拖拽式列映射
- 不做数据库导入导出模板生成

## 交互与页面

- 左侧：
  - 原始输入区
  - 输入格式切换：`CSV` / `JSON`
  - 分隔符与“是否有表头”设置
  - 模板按钮
- 右侧：
  - tabs：`表格预览` / `JSON 输出` / `CSV 输出`
  - 行列统计
  - 错误提示或解析成功提示
- 操作：
  - 格式化
  - 清空
  - 复制结果
  - 用输出覆盖输入

## 数据与状态

- 本地存储：
  - 保存最近一次输入内容
  - 保存输入格式、分隔符、是否有表头
  - 保存当前输出 tab
- 全部在浏览器本地解析与转换

## 验收标准

- 输入合法 CSV：
  - 能正确识别列数与行数
  - 能展示基础表格预览
  - 能导出 JSON 数组
- 输入合法 JSON 数组：
  - 能转换成结构正确的 CSV
- 输入不合法内容：
  - 能给出清晰错误提示
- 用户可以复制 CSV 或 JSON 输出
- 切换分隔符和表头设置后，结果会即时更新

## 风险与待确认问题

- CSV 边界情况很多，首版需要明确是否支持引号包裹、转义逗号、换行字段
- JSON → CSV 时，嵌套对象和数组如何展平需要定规则，首版建议只支持一层对象
- 表格预览如果直接渲染超大内容，性能可能受影响，后续可考虑添加预览行数上限
- 后续可扩展：
  - 列筛选与排序
  - 列名重命名
  - 文件导入导出
  - TSV 专项体验优化
