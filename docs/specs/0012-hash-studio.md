# 0012 Hash Studio

## 背景

文件校验、文本指纹、下载完整性验证、临时生成签名摘要，都是开发、安全、测试甚至普通办公场景里会碰到的需求。做一个本地 Hash 工具，不仅有开发价值，也有更广泛的“百宝箱”属性。

## 用户故事

作为开发者、测试或需要校验文件完整性的用户，我希望输入文本或选择文件后，立刻得到多种哈希结果并能一键复制，从而快速做校验、比对或记录。

## 本次范围

- 支持文本输入和文件选择
- 计算 SHA-1、SHA-256、SHA-512、MD5（如浏览器能力允许或使用轻量实现）
- 展示 hash 长度与摘要
- 支持不同算法结果一键复制
- 支持比较两个 hash 是否一致

## 不做什么

- 不做密码学签名
- 不做远程文件抓取
- 不做大文件断点校验和流式上传

## 交互与页面

- 左侧输入文本或上传文件
- 右侧显示各算法结果和校验状态
- 提供“目标 hash”输入框做快速比对

## 数据与状态

- 输入来源：用户本地文本或文件
- 输出结果：浏览器本地计算的 hash
- 是否需要本地存储：不保存文件，仅可保存最近一次文本输入

## 验收标准

- 输入文本后能实时看到多种 hash 结果
- 上传文件后也能正常计算
- 目标 hash 比对结果明确
- 复制功能可用

## 风险与待确认问题

- 原生 Web Crypto 对 MD5 不支持，需要评估是否接入轻量补充实现
- 大文件计算要注意 UI 卡顿
- 文件模式和文本模式的状态切换要足够清晰
