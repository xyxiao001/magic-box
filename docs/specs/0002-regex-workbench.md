# 0002 Regex Workbench

## 背景

程序员在处理日志、接口文本、表单校验和批量替换时，经常需要一个快速验证正则表达式的工具。相比跳转到外部站点，内置一个轻量但顺手的正则工作台更符合 `Magic Box` 的定位。

## 用户故事

作为开发者，我希望输入正则、测试文本和标志位后，能立即看到匹配结果、分组信息和替换预览，从而更快完成调试。

## 本次范围

- 支持输入正则表达式、测试文本和 flags
- 实时展示匹配结果
- 展示捕获分组
- 支持替换预览
- 提供常用示例

## 不做什么

- 不做正则语法教程全文档
- 不做多语言正则引擎差异模拟
- 不做云端保存

## 交互与页面

- 左侧输入正则、flags 和替换表达式
- 右侧展示匹配结果、命中列表和替换预览
- 支持复制结果

## 数据与状态

- 输入来源：用户手输
- 输出结果：浏览器本地实时计算
- 是否需要本地存储：可以后续增加最近使用记录

## 验收标准

- 可输入标准 JS 正则表达式并正确匹配
- 捕获组信息展示正确
- 替换预览结果正确
- 无效正则会明确报错

## 风险与待确认问题

- 复杂正则在大文本下的性能
- 多行文本高亮展示方式是否需要单独组件
