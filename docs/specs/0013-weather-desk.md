# 0013 Weather Desk

## 背景

天气工具看似简单，但它覆盖了通勤、出行、活动安排、拍摄、跑步、差旅行程等大量日常场景。相比“去搜索天气”，一个更轻、更快、可切换城市、可查看未来几天趋势的工具，更符合百宝箱产品的即时使用习惯。

## 用户故事

作为普通用户、通勤者或经常出差的人，我希望快速查看当前位置或指定城市的天气、温度、降雨和未来趋势，从而更快决定今天怎么出门、要不要带伞、适不适合安排行程。

## 本次范围

- 支持城市搜索和手动切换城市
- 支持当前位置天气（用户授权后）
- 展示当前温度、体感、天气状况、风力、湿度
- 展示未来 24 小时或未来几天趋势
- 支持常用城市收藏

## 不做什么

- 不做复杂灾害预警系统
- 不做地图气象图层
- 不做账号同步和长期历史统计

## 交互与页面

- 左侧搜索城市或切换当前位置
- 右侧展示当前天气卡片和趋势卡片
- 收藏城市后可一键切换

## 数据与状态

- 输入来源：定位权限或手动搜索
- 输出结果：实时天气、未来趋势、城市列表
- 是否需要网络：需要第三方天气 API
- 是否需要本地存储：建议存最近城市和收藏城市

## 验收标准

- 输入城市后能看到天气结果
- 开启定位后能获取当前位置天气
- 趋势数据清晰可读
- 失败时要有明确错误提示

## 风险与待确认问题

- 天气数据依赖外部 API，需要选稳定且限制合理的服务
- 地区语言、城市重名和定位失败都要做好兜底
- 这是少数“不是纯本地”的工具，需要在产品层说明
