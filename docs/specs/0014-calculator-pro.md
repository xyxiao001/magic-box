# 0014 Calculator Pro

## 背景

计算器是最典型的高频小工具之一，但在真实场景里，人们不只做加减乘除，还会临时算百分比、折扣、税率、汇总、日期差、分摊金额。做一个带表达式、历史记录和常用模式的计算器，能极大增强百宝箱的“随手可用”属性。

## 用户故事

作为普通用户、运营、财务、产品或开发者，我希望快速输入表达式并立刻得到结果，还能查看历史和常见计算场景，从而减少切换到系统计算器或搜索引擎的频率。

## 本次范围

- 支持基础四则运算与括号
- 支持百分比、取整、小数格式化
- 支持表达式输入与即时结果
- 支持历史记录
- 提供几个快捷模式：折扣、税率、平均分摊

## 不做什么

- 不做科学计算器
- 不做图形函数
- 不做复杂财务模型

## 交互与页面

- 左侧输入表达式或选择快捷模式
- 右侧展示结果、过程摘要与历史记录
- 常见结果支持复制

## 数据与状态

- 输入来源：用户键盘输入或按钮输入
- 输出结果：计算结果和历史记录
- 是否需要网络：不需要
- 是否需要本地存储：建议保存最近历史

## 验收标准

- 普通表达式能正确计算
- 常见模式输入后结果清晰
- 历史记录可回看
- 非法表达式有明确提示

## 风险与待确认问题

- 表达式解析要避免 `eval` 类实现
- 小数精度与百分比语义要定义清楚
- 快捷模式不能做得太杂，否则首版会失焦
