# 0004 QRCode Studio

## 背景

二维码生成是典型高频轻工具，适合放进百宝箱里作为快速交付模块，也适合作为后续分享、下载和视觉生成类工具的切入点。

## 用户故事

作为开发者或内容运营，我希望输入链接或任意文本后，立即生成二维码并下载，从而快速用于测试、分享和打印。

## 本次范围

- 输入文本或 URL 生成二维码
- 支持尺寸切换
- 支持前景色和背景色基础配置
- 支持下载 PNG

## 不做什么

- 不做复杂品牌模版系统
- 不做批量二维码生成
- 不做扫码统计

## 交互与页面

- 左侧配置文本、尺寸和颜色
- 右侧实时预览二维码
- 提供下载按钮

## 数据与状态

- 输入来源：用户手输
- 输出结果：浏览器本地生成
- 是否需要本地存储：不是必须

## 验收标准

- 输入内容后可立即生成二维码
- 可切换尺寸并正确下载
- 配色切换后预览正确

## 风险与待确认问题

- 选择二维码生成库还是纯前端绘制
- 下载清晰度和移动端适配
