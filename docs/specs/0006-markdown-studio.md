# 0006 Markdown Studio

## 背景

Markdown 是程序员和内容团队都高频使用的格式，常见需求包括实时预览、模板起稿、代码块整理和快速复制。把它做成百宝箱里的本地优先工具，能很好补足当前工具矩阵。

## 用户故事

作为开发者或技术写作者，我希望一边编辑 Markdown 一边实时预览，并能快速套用常用模板，从而更高效地写说明文档、README 或发布内容。

## 本次范围

- 左侧编辑 Markdown，右侧实时预览
- 内置几类基础模板：README、Issue、更新日志、接口说明
- 支持一键复制原文
- 支持基础统计：字符数、段落数、代码块数

## 不做什么

- 不做多人协作
- 不做云端同步
- 不做复杂导出格式转换

## 交互与页面

- 左侧为编辑器，顶部可切换模板
- 右侧为渲染预览区
- 底部或顶部显示轻量统计信息

## 数据与状态

- 输入来源：用户本地编辑
- 输出结果：浏览器本地渲染
- 是否需要本地存储：建议保存当前草稿

## 验收标准

- 输入 Markdown 后预览实时更新
- 模板切换可正常带入默认内容
- 复制功能可用
- 基础统计信息展示正确

## 风险与待确认问题

- 是否引入 Markdown 渲染库需要评估包体积和样式控制成本
- 代码块高亮是否首版就做需要确认
- 预览区样式需要和整体 Liquid Glass 风格协调
